# Stupid Insult Discord Bot

# Summary
Discord bot that insults your friends based on what they say in Discord. Farily mean so be weary.

# How to use
Add to your discord server [here](https://discord.com/oauth2/authorize?client_id=1521280845384515748). Commands are "!computahinsult @{user}", which insults the user (may take a few seconds) and
"!computahforget @{user}", which allows you to remove all of your messages from the database. At first, it has no info on anyone to fuel insults, so it may need some time to collect messages to strike 
closer to the heart.

# How it Works
Very simple, just collects the last 50 messages of users in a discord server through c# events from the discord NuGet package and puts them in an SQLite database to fuel insults generated by Google Gemini.

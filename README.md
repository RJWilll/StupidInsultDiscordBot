# Stupid Insult Discord Bot

# Summary
Discord bot that insults your friends based on what they say in Discord. Farily mean so be weary.

# How to use
Add to your discord server [here](https://discord.com/oauth2/authorize?client_id=1521280845384515748) and use commands when bot is online (no promises).

## Commands 
"!computahinsult @{user}", which insults the user (may take a few seconds) <br>
"!computahforget @{user}", which allows you to remove all of your messages from the database for a message reset <br>
"!computahblock @{user}", which forgets all user's messages, blocks user from being insulted, and prevents any further message collection from that user <br>
"!computahunblock @{user}", which unblocks user, resuming message collection, and allowing them to be insulted <br>
<br>
At first, it has no info on anyone to fuel insults, so it may need some time to collect messages to strike closer to the heart. 

# How it Works
Very simple, just collects the last 50 messages of users in a discord server through c# events from the discord NuGet package and puts them in an SQLite database to fuel insults generated by Google Gemini.
